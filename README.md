# Amazon Clone
I ran a Amazon clone with React. 

### Used
    FRONTEND
        ReactJS + Hooks
        React Context Api
        Material UI
        Firebase - Hosting, Firestore Database, Authentication
        Payment - stripe
    
    BACKEND
        NodeJS + Express Server
        Firebase Functions
